# Shell Scripts

### blind_sqli.sh
This is a scripted Blind SQL injection that saves time by automating the attack process.

### dns3num.sh
A simple script I wrote to perform DNS recon on a target.

### hostFind3r.sh
A script that is useful when hunting subdomains of a target.

### ping_sweep.sh
Based on Georgia Weidman's script from her Penetration Testing Book. Performs a ping sweep on a /24 network.

### r3con.sh
Based on TheCyberMentor's web recon script. Used to automate the recon process for Bug Bounty programs.

### reverseFind3r.sh
Performs a reverse DNS lookup of a host.

### start.sh
Useful for automating redundant tasks when working through CTF challenges. It creates a directory with the target's name and adds the /nmap and /gobuster dirs within it.
